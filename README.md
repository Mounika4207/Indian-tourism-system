# 🇮🇳 Indian Tourism Project

## 📌 Description
This project is a web application that showcases popular tourist places in India.

## 🚀 Features
- Explore tourist destinations
- View images and details
- Responsive design

## 🛠️ Technologies Used
- HTML
- CSS
- JavaScript

## ▶️ How to Run
1. Download the project
2. Open index.html in browser
